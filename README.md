# CPP Storage Engine

Простой in-memory key-value storage движок с асинхронной записью на диск и поддержкой Linux системных вызовов.

Проект реализует базовые принципы работы storage-систем: хранение данных в памяти, журналирование операций (WAL), восстановление состояния и измерение производительности.

---

# 🚀 Возможности

- Хранение данных в памяти (`unordered_map`)
- Потокобезопасные операции (`shared_mutex`)
- Асинхронная запись операций (очередь + worker thread)
- Журналирование операций (WAL-подобная модель)
- Использование Linux системных вызовов (`open`, `write`)
- Восстановление состояния из файла (`load`)
- Простое CLI управление
- Бенчмарк производительности (PUT/GET операций)

---

---

# ⚙️ Команды CLI

- `PUT key value` — добавить/обновить значение
- `GET key` — получить значение
- `DEL key` — удалить ключ
- `CLEAR` — очистить хранилище
- `ALL` — показать все данные
- `BENCH` — запустить тест производительности
- `EXIT` — выход

Сбор проекта : 
- cd build
- cmake ..
- make
- ./storage

---

# 📊 Бенчмарк

Проект включает простой тест производительности:

- PUT операций в секунду
- GET операций в секунду
- измерение времени выполнения

# 🧱 Используемые технологии

- C++17

  - open()
  - write()
  - close()
- Многопоточность (std::thread)
- Синхронизация (mutex, shared_mutex)
- CMake
